Class: Character

Character()

Character The main character of the game (Pepe). Contains all image arrays for different animation states. Extends the MovableObject class.

Constructor

new Character()

Creates a new Character instance and initializes its animation states. Loads all relevant image sequences, applies gravity and movement, and sets the initial state to "idle".
